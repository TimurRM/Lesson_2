## Lists

For adding unnumerated lists you need to highlite items with asterisk (*). 

Example:
* Element 1
* Element 2
* Element 3

For adding numerated lists you need to highlite items with numbers (1.). 

Example:
1. Element 1
2. Element 2
3. Element 3
